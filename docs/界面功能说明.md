# SortFlow 页面功能说明（现状梳理）

> 本文档用于完整记录当前界面所有功能点，便于后续让 AI 重新设计界面时不遗漏任何交互与信息。

## 1. 全局框架与导航结构
- **三大页面/标签切换**：应用通过 `dashboard / config / history` 三个标签页切换渲染内容。
- **整体布局结构**：
  - 左侧可折叠主导航栏（桌面端显示，移动端隐藏）。
  - 顶部应用栏包含搜索框、任务状态入口、通知入口。
  - 主内容区随标签切换内容。
  - Dashboard 下右侧显示“Transaction Desk”执行面板。

## 2. 左侧主导航栏（Layout Sidebar）
- **品牌区**：Logo + 应用名 + 版本标识（SortFlow v2.0）。
- **三大导航按钮**：Dashboard / Configuration / History Log。
  - 当前激活项高亮背景。
  - 折叠时仅显示图标，展开时显示文本。
- **折叠按钮**：侧边栏右侧圆形“展开/收起”控制按钮，图标随状态变化。

## 3. 顶部应用栏（Header）
- **移动端品牌缩略**：小屏时显示 SortFlow 文本 + 图标。
- **搜索输入**：placeholder 为 “Search files, metadata…”，仅在中屏以上显示（移动端隐藏）。
- **任务状态入口**：
  - 任务按钮显示“pending_actions”图标。
  - 若有 active 任务，显示右上角橙色状态点。
  - 点击弹出“Task Status List”面板：
    - 列表展示任务名称、进度、已处理数、剩余时间。
    - 任务状态图标（执行中旋转、完成勾选）。
    - 进度条 + 百分比。
    - “View Full History”入口按钮。
- **通知按钮**：仅图标入口（无弹层实现）。

## 4. Dashboard 页面（主要工作台）

### 4.1 左侧 Source Watchers 树形目录
- 显示 `config.sourceWatchers` 列表；为空时提示 “No watchers configured”。
- 每个 watcher 作为顶层 TreeItem：
  - 支持展开/收起子目录。
  - 激活路径高亮。
  - 展开时加载子目录（通过后端 `fileService.listFiles`）。
  - loading 状态显示旋转图标。
  - 子目录递归呈现。
  - 点击目录会设置当前路径并展开/收起。

### 4.2 顶部路径栏
- 显示当前路径（`currentPath`）。
- 路径前带图标，作为主内容区标题栏。

### 4.3 Pattern Mixer（批量命名规则）
- **RAW_NAME 选项**：勾选保持原始文件名，清空 token/keyword 组合。
- **临时关键字输入**：
  - 输入后加入命名规则。
  - 提示“发起迁移任务后会自动清空”。
- **Token 选择区（基于文件名拆分）**：
  - 从已选文件名提取 tokens。
  - 点击 token 进入组合，再次点击取消。
  - 选中顺序影响命名顺序。
- **Custom Keywords（来自配置）**：
  - `config.customKeywords` 与 `keywords` 两类来源。
  - 点击切换选中状态，选中时高亮。

### 4.4 File List（文件网格）
- **过滤逻辑**：
  - 仅展示当前路径文件。
  - 可隐藏非媒体文件（由 `config.hideNonMedia` 控制）。
- **排序功能**：
  - 排序字段：创建时间 / 更新时间 / 文件名。
  - 排序顺序：升序 / 降序。
- **多选模式**：
  - 显示“多选”按钮。
  - 开启后点击文件保持多选行为。
- **文件卡片展示**：
  - 缩略图或图标占位。
  - 标签显示文件类型。
  - 选中状态显示勾选标记。
  - 展示文件名、大小与类型。
- **交互**：
  - 单击选择（支持 Ctrl/Cmd 或 Shift 多选）。
  - 双击目录进入子目录。
  - 双击媒体文件打开预览。
  - 空格键打开已选媒体预览。

### 4.5 媒体预览覆盖层（Media Preview）
- 展示图片或视频。
- ESC 关闭。
- 空格播放/暂停视频。
- 点击遮罩关闭。
- 预览时禁止页面滚动。

### 4.6 右侧 Transaction Desk（执行面板）
- **队列统计**：显示当前预览队列数量。
- **目标根目录选择**：
  - 列表展示 target roots。
  - 选择后高亮。
  - 显示名称、路径、图标。
- **Organizational Class（预设选择）**：
  - 展示 preset 分类。
  - 点击切换，高亮选中项。
- **最终路径预览**：显示目标路径拼接结果（target + preset）。
- **队列预览列表**：显示原始文件名 → 新文件名。
- **执行按钮**：
  - 文案 “COMMIT TO ARCHIVE”。
  - 队列为空或预览加载中禁用。
  - 点击触发执行流程。

### 4.7 执行状态浮层（右下角）
- 执行中显示进度。
- 完成或失败显示结果。
- 失败显示错误信息。
- 完成/失败后支持“关闭”。

## 5. Configuration 页面

### 5.1 顶部区域
- 标题/说明 + “Refresh” 按钮重新拉取配置。

### 5.2 Display 设置
- “仅显示媒体文件（图片/视频）”开关，对应 `config.hideNonMedia`。

### 5.3 Pattern Mixer 自定义关键字
- 可添加/删除关键字。
- 数据保存到 `localStorage`，同步到 `config.customKeywords`。
- 点击关键字可移除。

### 5.4 Video Covers
- “Generate Video Covers” 按钮。
- 扫描所有 watcher 生成视频封面。
- 显示生成结果统计。

### 5.5 Source Watchers
- 列表展示 watcher 路径。
- 支持移除。
- 支持输入新路径并添加。

### 5.6 Target Roots
- 列表展示目标根目录（名称/路径/图标）。
- 支持移除。
- 表单新增：名称 / 路径 / icon 字段。

### 5.7 Mixer Keywords（后台关键字）
- 列表展示关键词。
- 可编辑并保存。
- 可删除。
- 支持新增关键词。

### 5.8 Presets（规则预设）
- 列表展示 preset（名称/子路径/前缀/图标）。
- 支持删除。
- 表单新增：Preset Name / 子路径 / Default Prefix。

## 6. History Log 页面
- 标题、说明 + Refresh 按钮重新拉取历史记录。
- 空状态提示 “History Log is empty”。
- 列表展示：
  - 操作类型
  - 时间戳
  - 文件数
  - 状态（completed / failed）

## 7. 执行逻辑与数据流（Dashboard 核心行为）
- **配置加载**：
  - 初始拉取 config / presets / targets / keywords。
  - 失败回退到本地常量。
  - 关键字从 `localStorage` 读取合并。
- **文件加载**：根据当前路径加载文件；失败回退到常量数据。
- **执行任务**：
  - 根据选中文件 + mixer 配置 + preset/target 生成预览操作。
  - 组装 move actions，调用执行服务。
  - 通过 WebSocket 监听执行状态更新（进度、完成、错误）。
