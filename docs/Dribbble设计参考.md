# Dribbble 上查找类似工具设计图的方式

> 目标：更快找到与 SortFlow 类似的「文件管理 / 媒体整理 / 仪表盘」类工具界面设计参考，并建立可复用的检索方法。

## 1) 搜索关键词（组合检索）
把「功能域」+「界面类型」组合成检索词，命中率更高：

- 功能域关键词：
  - `file management` / `file manager`
  - `media manager` / `media library`
  - `asset manager` / `digital asset management`
  - `organizer` / `archive` / `workflow`

- 界面类型关键词：
  - `dashboard` / `admin` / `control panel`
  - `workspace` / `productivity` / `tool`
  - `table` / `grid` / `list` / `card`

**示例组合**：
- `file management dashboard`
- `media library admin panel`
- `asset manager workspace`
- `digital asset management ui`

## 2) 利用标签（Tags）与筛选（Filters）
- **标签**：搜索后进入结果页，留意标签（例如：`Dashboard`、`Admin`、`SaaS`、`Productivity`），点击标签可进一步收窄范围。
- **筛选**：
  - **Shots / Collections**：Shot 适合找单页布局，Collection 适合找成套界面风格。
  - **Time**：过滤到最近一年，减少过时设计。

## 3) 反向检索：从已命中作品到更多相似作品
- 打开一个看起来接近的作品，继续浏览：
  - **作者主页**：往往会有同风格/同产品线的一组界面。
  - **More by Designer**：设计师通常在同类型产品上有延续。
  - **Collections**：查看该作品是否被收录到某个集合里，从集合中挖掘相似主题。

## 4) 关注对象建议（风格方向）
> 目的：当你找到一个喜欢的视觉方向时，可以从设计师/工作室持续获得相似作品。

- **偏 SaaS / 管理后台**：搜索 `SaaS dashboard` / `admin panel`，进入作品作者主页，关注其后续作品。
- **偏极简/高效率工具**：搜索 `productivity tool` / `workspace ui`。
- **偏数据/流程导向**：搜索 `workflow dashboard` / `data management`。

## 5) 结合 SortFlow 的关键词建议
SortFlow 具有「文件列表 + 规则组合 + 执行面板」的典型工作流特征，建议：

- `file explorer ui`
- `batch rename tool ui`
- `content management dashboard`
- `media organization tool`
- `workflow dashboard ui`

## 6) 快速收藏与整理方式
- **创建 Collection**：把命中的界面统一收藏，后续和 AI 设计师对齐参考风格。
- **截图 + 标注**：截取布局结构（左侧导航 / 中部列表 / 右侧执行区），标注关键层级，便于 AI 复刻信息架构。

## 7) 若找不到足够相似的作品
可使用「替代域」进行检索：
- `project management dashboard`
- `inventory management ui`
- `document management system`
- `video library dashboard`

这些领域常有类似的「列表 + 操作面板 + 右侧确认区」布局。
